Pengertian HTML

HTML (HyperText Markup Language) adalah bahasa pemrograman standar yang digunakan untuk membuat susunan struktur dasar sebuah halaman website. Ibarat membangun sebuah rumah, HTML bertindak sebagai dinding, tiang, dan kerangka atapnya sebelum rumah tersebut dicat atau dihias. Melalui dokumen HTML, kita dapat menampilkan teks, menyisipkan gambar, hingga menghubungkan satu halaman dengan halaman lainnya di internet.

Element HTML

Elemen HTML adalah komponen penyusun dokumen web yang terdiri dari tag pembuka, isi konten di dalamnya, serta diakhiri dengan tag penutup. Setiap elemen memberi tahu browser bagaimana cara menampilkan konten tersebut kepada pengguna secara visual. Mengetahui cara menulis elemen dengan benar adalah kunci utama dalam menguasai HTML secara keseluruhan.

Struktur Element:
<tagname>Konten Anda disini...</tagname>

Heading HTML

HTML memiliki enam tingkatan heading atau judul halaman yang didefinisikan dengan tag <h1> hingga <h6>. Elemen <h1> digunakan untuk judul utama yang paling penting dan berukuran paling besar, sementara elemen <h6> digunakan untuk sub-judul yang paling tidak penting dengan ukuran terkecil.

Syntax Kode:

          <h1>Judul Tingkat 1</h1> 

          <h2>Judul Tingkat 2</h2> 

          <h3>Judul Tingkat 3</h3> 

          <h4>Judul Tingkat 4</h4> 

          <h5>Judul Tingkat 5</h5> 

          <h6>Judul Tingkat 6</h6>

Paragraf HTML

Paragraf dalam HTML selalu diawali dengan baris baru dan didefinisikan menggunakan tag <p>. Browser web secara otomatis akan menambahkan ruang kosong atau margin sebelum dan sesudah paragraf tersebut agar teks nyaman dibaca oleh pengguna di layar gadget.

Syntax Kode:
<p>Ini adalah sebuah paragraf teks.</p>
Hasil:

Ini adalah sebuah paragraf teks.

Kutipan HTML

Elemen kutipan digunakan untuk menandai bagian teks yang diambil langsung dari sumber atau pernyataan orang lain. HTML menyediakan tag <blockquote> untuk kutipan panjang berbentuk blok paragraf tersendiri, dan tag <q> untuk kutipan pendek yang berada di dalam baris kalimat biasa lengkap dengan tanda petik otomatis.

Syntax Kode:
<p>Dia berkata,
<q>Tetap semangat belajar!</q>
</p>
Hasil:

Dia berkata, Tetap semangat belajar!

Komentar HTML

Komentar HTML adalah baris kode catatan yang sengaja disisipkan untuk membantu developer memahami alur struktur kode pemrograman. Bagian komentar yang diawali dengan penanda tertentu ini tidak akan diproses atau ditampilkan oleh browser kepada pengguna umum di halaman depan website Anda.

Syntax Kode:
<!-- Komentar ini tidak akan tampil di browser-->

Color HTML

Penulisan warna pada elemen HTML dapat dikonfigurasikan dengan menggunakan nama warna bahasa Inggris yang terstandarisasi, ataupun menggunakan format nilai kode seperti HEX, RGB, dan HSL. Warna ini bisa diterapkan langsung pada teks maupun sebagai latar belakang kotak elemen.

Syntax Kode:
<p style="color:#ff6b81;">
Teks berwarna pink
</p>
Hasil:

Teks berwarna pink

Link HTML

Tautan atau hyperlink dibuat menggunakan tag jangkar yaitu <a>, yang berfungsi menghubungkan pengguna ke alamat halaman web eksternal maupun internal. Atribut paling krusial di dalam tag ini adalah href yang mendefinisikan target URL tujuan ketika teks tautan tersebut diklik.

Syntax Kode:
<a href="#">Klik Link Ini</a>
Hasil:
Klik Link Ini

Gambar HTML

Untuk menyisipkan ilustrasi gambar ke dalam halaman web, kita menggunakan elemen tunggal <img> tanpa tag penutup. Elemen ini mengandalkan atribut src yang mengarah pada direktori file gambar, serta atribut alt untuk menampilkan teks deskripsi cadangan jika gambar gagal dimuat.

Syntax Kode:
<img src="logo.png" alt="Logo Belajar" width="100">

Favicon HTML

Favicon merupakan ikon atau logo gambar berukuran sangat kecil yang muncul tepat di samping teks judul halaman pada tab browser internet Anda. Elemen favicon ini umumnya dikonfigurasikan di dalam bagian kepala dokumen web atau tag khusus berbentuk tautan relasi icon.

Syntax Kode:
<link rel="icon" type="image/x-icon" href="favicon.ico">

Tabel HTML

Pembuatan susunan baris data kotak terstruktur dilakukan melalui tag pembungkus utama <table>. Di dalamnya terdapat elemen baris <tr>, elemen judul kolom tabel yang tebal <th>, serta baris kolom pengisian sel data informasi normal yaitu tag <td>.

Syntax Kode:

<tr>
<th>Menu</th>
</tr>
<tr>
<td>Data HTML</td>
</tr>
</table>

Div HTML

Tag pembungkus generic <div> digunakan sebagai wadah penampung berkelompok untuk menyatukan beberapa elemen HTML sekaligus menjadi satu kesatuan blok. Elemen ini sangat berguna dalam mempermudah penataan tata letak layout ataupun membagi area halaman situs sebelum dihias memakai CSS.

Syntax Kode:
<div>
<h2>Judul di dalam Div</h2>
<p>Paragraf ini berada di dalam satu wadah pembungkus.</p>
</div>

Class HTML

Atribut class digunakan untuk memberikan label nama penanda identitas kelompok pada satu atau beberapa elemen HTML secara massal. Dengan menggunakan kelas, Anda dapat mengatur gaya desain tampilan atau memprogram banyak baris elemen yang berbeda secara serentak dari file eksternal.

Syntax Kode:
<p class="teks-pink">
Paragraf pertama berwarna pink.
</p>
<p class="teks-pink">
Paragraf kedua juga berwarna pink.
</p>

ID HTML

Atribut id berfungsi menetapkan pengenal atau identitas unik yang mutlak bersifat tunggal dan khusus untuk satu elemen HTML saja di dalam satu halaman web. Nilai id dilarang keras memiliki nama kembar, karena digunakan sebagai target utama penataan CSS spesifik atau manipulasi JavaScript.

Syntax Kode:
<div id="header-utama">
<h1>Selamat Datang di Basic CoderEdu</h1>
</div>

Pengertian CSS

CSS (Cascading Style Sheets) adalah bahasa pemrograman desain yang digunakan untuk mengatur dan mengontrol tampilan visual halaman website yang dibuat menggunakan HTML. Jika HTML diibaratkan sebagai struktur tulang atau dinding batako dari sebuah rumah, maka CSS bertindak sebagai cat dinding, dekorasi interior, gorden, tata letak lampu, hingga penataan furnitur agar rumah tersebut terlihat indah, rapi, dan estetis.

Dengan menggunakan CSS, Anda dapat memisahkan konten (HTML) dari desain (CSS). Hal ini mempermudah pengelolaan web karena Anda bisa mengubah tampilan seluruh halaman website hanya dengan memodifikasi satu file CSS saja.

Sintaks CSS

Aturan penulisan (sintaks) CSS sangat sederhana dan terdiri dari dua komponen utama, yaitu: Selector dan Declaration Block (Blok Deklarasi).

Selector berfungsi untuk menunjuk elemen HTML mana yang ingin dihias, sedangkan Blok Deklarasi (dimulai dengan { dan diakhiri }) berisi satu atau beberapa deklarasi gaya yang dipisahkan oleh tanda titik koma (;). Setiap deklarasi terdiri dari Property (nama efek) dan Value (nilai efek) yang dipisahkan tanda titik dua (:).

Aturan Struktur Sintaks:
selector {
property: value; property: value;
}

Element Selector CSS

Element Selector digunakan untuk memilih dan mengatur gaya seluruh elemen HTML berdasarkan nama tag-nya secara langsung (seperti <p>, <h1>, atau <button>). Semua elemen dengan nama tag yang sama di halaman tersebut akan otomatis mengikuti aturan desain yang telah ditentukan.

Syntax Kode CSS:
p {
color: #ff4d6d; text-align: center;
}

Cara Menambahkan CSS ke HTML

Ada 3 metode utama untuk menyisipkan atau mengaplikasikan kode CSS ke dalam dokumen HTML:

  1. Inline CSS: Ditulis langsung di dalam tag HTML menggunakan atribut style.
  2. Internal CSS: Ditulis di dalam satu file HTML yang sama, diletakkan di dalam elemen <style> pada bagian <head>.
  3. External CSS: Ditulis di file terpisah berformat .css, lalu dihubungkan ke HTML menggunakan tag <link> di dalam bagian <head> (Metode paling direkomendasikan).
Contoh Kode (External CSS):
<head>
<link rel="stylesheet" href="style.css">
</head>

Komentar CSS

Komentar CSS adalah catatan atau penjelasan yang ditulis di dalam file CSS untuk membantu developer memahami alur penulisan kode. Komentar ini sepenuhnya diabaikan oleh browser dan tidak akan memengaruhi tampilan website Anda. Komentar CSS dimulai dengan tanda /* dan diakhiri dengan tanda */.

Syntax Kode CSS:
/* Ini adalah komentar satu baris */
p { color: black;
/* Mengatur warna teks menjadi hitam */
}
/* Ini adalah komentar yang ditulis lebih dari satu baris */

Color CSS

Syntax Kode CSS:
h3 {
color: tomato;
}
p {
color: #ff8fa3;
}
span {
color: rgb(255, 77, 109);
}

Property background digunakan untuk mengatur efek latar belakang dari suatu elemen HTML. Anda bisa mengatur warna latar belakang menggunakan background-color ataupun menyisipkan gambar sebagai latar belakang menggunakan properti background-image.

Syntax Kode CSS:

Border CSS

Property CSS Border memungkinkan Anda untuk menentukan garis tepi di sekitar elemen HTML. Properti ini membutuhkan tiga nilai utama: ketebalan garis (border-width), gaya garis seperti lurus/putus-putus (border-style), dan warna garis (border-color).

Syntax Kode CSS:
.kotak-border {
border-width: 2px;
border-style: dashed;
border-color: #ff4d6d;
}
Hasil Tampilan di Layar (Live Preview):
Kotak dengan garis tepi putus-putus (dashed border) berwarna pink!

Padding CSS

CSS Padding digunakan untuk membuat ruang kosong di dalam elemen, yaitu jarak antara konten utama (seperti teks atau gambar) dengan batas garis tepi (border) milik elemen itu sendiri. Padding sepenuhnya bersifat transparan dan akan ikut menduplikasi warna latar belakang dari elemen tersebut.

Syntax Kode CSS:
/* Jarak ruang di dalam kotak */
.kotak-padding {
border: 2px solid #ccc;
padding: 25px;
}
Hasil Tampilan di Layar (Live Preview):
Teks ini memiliki jarak longgar sebesar 25px dari garis tepi pink di sekelilingnya karena efek padding.

Margin CSS

Berbeda dengan padding, CSS Margin digunakan untuk membuat ruang kosong di luar elemen, yaitu mengatur jarak pemisah antara elemen tersebut dengan elemen-elemen lain yang berada di sekitarnya. Margin menciptakan jarak antar kotak secara horizontal maupun vertikal dan tidak memiliki warna latar belakang (selalu transparan).

Syntax Kode CSS:
.kotak-margin {
margin: 30px;
/* Jarak kotak dari luar */
border: 2px solid #333;
}
Visualisasi Efek Jarak Jauh Margin:

Outline CSS

Outline adalah garis tambahan yang digambar tepat di luar garis tepi (border) elemen untuk membuat elemen tersebut terlihat lebih menonjol atau fokus. Perbedaan utama outline dengan border adalah outline tidak memakan ruang tata letak halaman (tidak memengaruhi ukuran dimensi lebar/tinggi kotak elemen) dan digambar di luar batas margin/border.

Syntax Kode CSS:
.kotak-outline {
border: 2px solid black;
outline: 4px solid #ff4d6d;
}
Hasil Tampilan di Layar (Live Preview):
Kotak ini memiliki border hitam, dan di luar border hitam terdapat garis outline berwarna pink cerah!

Pengertian JavaScript

JavaScript (JS) adalah bahasa pemrograman tingkat tinggi, dinamis, dan bersifat client-side scripting. Jika HTML bertindak sebagai struktur bangunan website dan CSS sebagai desain tampilannya, maka JavaScript bertindak sebagai otak atau sistem penggerak yang membuat website tersebut menjadi hidup, interaktif, dan dinamis.

Dengan JavaScript, Anda dapat membuat fitur-fitur interaktif seperti tombol penampil pop-up, kalkulator, pembaruan konten secara *real-time* tanpa memuat ulang halaman (AJAX), animasi grafik, validasi formulir input, hingga logika permainan.

Cara Menambahkan JavaScript ke HTML

Ada dua cara utama yang paling sering digunakan untuk menyisipkan kode JavaScript ke dalam dokumen HTML Anda:

A. Internal JavaScript

Kode JavaScript ditulis langsung di dalam file HTML menggunakan tag <script>. Biasanya diletakkan di bagian paling bawah sebelum penutup tag </body> agar halaman web selesai dimuat terlebih dahulu.

Contoh Internal JS:
<script>
console.log("Halo dari Internal JS!");
</script>

B. External JavaScript

Kode JavaScript ditulis di dalam file terpisah dengan ekstensi .js, lalu dihubungkan ke file HTML menggunakan atribut src pada tag <script>. Ini merupakan metode terbaik agar kode tetap rapi.

Contoh Hubungan External JS:
<script src="script.js"></script>

Sintaks JavaScript

JavaScript memiliki beberapa aturan sintaksis penting. Untuk membuat variabel, kita dapat menggunakan kata kunci var, let, atau const. Setiap baris pernyataan opsional diakhiri dengan tanda titik koma (;), dan JavaScript bersifat Case-Sensitive (membedakan huruf besar dan huruf kecil, contoh: variabel nama berbeda dengan Nama).

Syntax Kode JavaScript:
let namaAplikasi = "Basic CoderEdu";
const versi = 2.0;
console.log(namaAplikasi + " Versi " + versi);
Hasil Output Konsol Browser:
Basic CoderEdu Versi 2.0

Operator JavaScript

Operator digunakan untuk melakukan operasi aritmatika (matematika) atau perbandingan logika pada variabel dan nilai data. Jenis operator yang paling sering digunakan meliputi operator aritmatika (+, -, *, /, %) dan operator penugasan (=).

Syntax Kode JavaScript:
let angka1 = 10;
let angka2 = 5;
let hasilTambah = angka1 + angka2;
let hasilKali = angka1 * angka2;
console.log("Hasil: " + hasilTambah + " dan " + hasilKali);
Hasil Output:
Hasil: 15 dan 50

Kondisi IF Else JavaScript

Pernyataan kondisional digunakan untuk melakukan tindakan aksi tindakan yang berbeda berdasarkan kondisi logika yang dievaluasi. Jika kondisi bernilai benar (true), maka blok kode di dalam if akan dieksekusi. Jika salah (false), sistem akan beralih ke blok else.

Syntax Kode JavaScript:
let nilaiUjian = 85;
if (nilaiUjian >= 75) {
console.log("Status: LULUS");
} else { console.log("Status: REMEDIAL");
}
Hasil Output:
Status: LULUS

Perulangan JavaScript

Perulangan sangat berguna untuk menjalankan baris kode yang sama secara berulang-ulang selama kondisi tertentu terpenuhi. Struktur perulangan yang paling umum digunakan adalah for loop.

Syntax Kode JavaScript:
for (let i = 1; i <= 3; i++) {
console.log("Perulangan ke-" + i);
}
Hasil Output:
Perulangan ke-1 Perulangan ke-2 Perulangan ke-3

Functions JavaScript

Fungsi adalah sekumpulan blok kode terorganisir yang dirancang khusus untuk melakukan tugas tertentu. Fungsi tidak akan berjalan sebelum ia dipanggil (*invoke*). Fungsi sangat membantu dalam mencegah penulisan kode yang berulang-ulang (*reusable code*).

Syntax Kode JavaScript:
function sapaPengguna(username) {
return "Selamat datang kembali, " + username + "!";
}
// Memanggil fungsi dan menyimpannya ke variabel
let pesan = sapaPengguna("SiswaEduPink");
console.log(pesan);
Hasil Output:
Selamat datang kembali, SiswaEduPink!

Class JavaScript

Sejak pembaruan ECMAScript 6 (ES6), JavaScript mendukung pembuatan struktur kelas bertipe OOP (Object-Oriented Programming). Class bertindak sebagai cetakan blueprint untuk membuat objek konkret yang memiliki properti awal lewat metode khusus bernama constructor().

Syntax Kode JavaScript:
class Mobil {
constructor(merek) {
this.namaMobil = merek;
}
}
let mobilSaya = new Mobil("Toyota");
console.log("Mobil saya adalah " + mobilSaya.namaMobil);
Hasil Output:
Mobil saya adalah Toyota

Array JavaScript

Array adalah variabel khusus berjenis objek yang mampu menyimpan lebih dari satu nilai data dalam satu waktu sekaligus dengan tipe data yang bisa berbeda-beda. Nilai di dalam Array diakses berdasarkan nomor urut urutan indeksnya yang dimulai dari angka 0.

Syntax Kode JavaScript:
let bahasaWeb = ["HTML", "CSS", "JavaScript"];
console.log("Saya sedang belajar " + bahasaWeb[2]);
Hasil Output:
Saya sedang belajar JavaScript

Pengertian Java

Java adalah bahasa pemrograman tingkat tinggi yang bersifat object-oriented (berorientasi objek) dan berbasis class. Salah satu slogan paling terkenal dari Java adalah "Write Once, Run Anywhere" (WORA), yang berarti kode program Java yang telah dikompilasi dapat dijalankan di berbagai sistem operasi (Windows, Mac, Linux) tanpa perlu dikompilasi ulang, berkat adanya komponen bernama Java Virtual Machine (JVM).

Print Hello World di Java

Untuk mencetak teks atau string ke layar dalam pemrograman Java, kita wajib menuliskannya di dalam metode utama kelas. Perintah yang digunakan untuk memproses instruksi pencetakan ini adalah fungsi bawaan Java.

Syntax Kode Java:
public class Main {
public static void main(String[] args) {
System.out.println("Hello World");
}
}
Hasil Tampilan Output di Terminal:
Hello World

Syntax Dasar Java

Setiap baris kode program Java yang dieksekusi harus selalu berada di dalam sebuah wadah bernama class. Nama class tersebut harus ditulis sama persis dengan nama file fisik berkas penyimpanannya (sensitif terhadap huruf besar dan kecil). Selain itu, setiap baris pernyataan instruksi wajib diakhiri dengan tanda titik koma (;).

Contoh Aturan Struktur:
// Nama kelas harus sama dengan nama file
Main.java public class Main {
// Fungsi Utama
public static void main(String[] args) {
// Pernyataan kode diakhiri dengan semicolon (;)
System.out.println("Belajar Syntax Java");
}
}
Output Terminal:
Belajar Syntax Java

Output Java

Jika kita ingin memproses atau menampilkan suatu nilai output data ke layar, kita wajib menggunakan metode perintah dari class sistem yaitu System.out.println() untuk cetak baris baru otomatis, atau opsi lainnya yaitu System.out.print() jika ingin mencetak teks tanpa tambahan baris baru setelahnya.

Syntax Kode:
System.out.print("Satu, ");
System.out.println("Dua");
System.out.println("Tiga");
Output Terminal:
Satu, Dua Tiga

Komentar Java

Komentar digunakan untuk memberikan catatan dokumentasi internal kode agar mudah dibaca oleh programmer lain. Baris komentar sepenuhnya akan diabaikan oleh compiler Java saat program dijalankan. Java mendukung komentar satu baris menggunakan tanda garis miring ganda (//) dan komentar multi-baris menggunakan kombinasi tanda (/* ... */).

Syntax Kode:
// Ini komentar satu baris
/* Ini komentar yang panjang atau lebih dari satu baris */
System.out.println("Komentar tersembunyi!");
Output Terminal:
Komentar tersembunyi!

Variabel Java

Variabel adalah wadah penyimpanan sementara memori komputer untuk mengamankan nilai data selama program dijalankan. Untuk membuat sebuah variabel baru di Java, kita wajib mendeklarasikan jenis tipe datanya terlebih dahulu sebelum menuliskan nama variabel beserta nilai datanya.

Syntax Kode:
String nama = "EduPink";
int angka = 25;
System.out.println(nama + " berumur " + angka);
Output Terminal:
EduPink berumur 25

Tipe Data Primitive

Tipe data primitif adalah tipe data bawaan orisinal dari Java yang menentukan ukuran dan jenis nilai data dasar secara langsung, serta tidak memiliki metode fungsi objek tambahan bawaan sendiri. Terdiri dari 8 macam jenis utama.

Contoh Penggunaan:
int angkaBulat = 1000;
double angkaDesimal = 19.99;
char inisial = 'A';
boolean status = true;
System.out.println(angkaBulat + " | " + angkaDesimal + " | " + inisial + " | " + status);
Output Terminal:
1000 | 19.99 | A | true

Tipe Data Non-Primitive

Tipe data non-primitif juga biasa disebut sebagai tipe data referensi (*Reference Types*) karena merujuk atau mengarah pada objek instansiasi tertentu. Tipe data ini dibuat oleh programmer dan memiliki banyak metode fungsi internal yang bisa dipanggil, contoh paling populer adalah tipe data objek teks kumpulan karakter teks (*String*).

Contoh Penggunaan:
String teksObjek = "Halo Dunia Java";
System.out.println(teksObjek.toUpperCase());
// Memanggil metode bawaan
Output Terminal:
HALO DUNIA JAVA

Operator Java

Operator adalah sekumpulan simbol matematika khusus yang digunakan untuk menjalankan operasi kalkulasi matematis, manipulasi logika, ataupun perbandingan antar nilai data variabel di dalam program Java.

Syntax Kode:
int x = 10;
int y = 5;
System.out.println("Hasil Tambah: " + (x + y));
System.out.println("Apakah Lebih Besar? " + (x > y));
Output Terminal:
Hasil Tambah: 15 Apakah Lebih Besar? true

Seleksi Kondisi Java

Blok penyeleksian kondisi digunakan untuk mengontrol aliran jalannya program. Pernyataan di dalam blok kode if hanya akan diproses dan dieksekusi jika kondisi syarat matematika yang diuji menghasilkan nilai kebenaran berstatus benar (*true*). Jika salah, program beralih ke blok kode else.

Syntax Kode:
int nilai = 75;
if (nilai >= 70) {
System.out.println("Lulus Kuliah!");
} else {
System.out.println("Remedial.");
}
Output Terminal:
Lulus Kuliah!

Switch Case Java

Pernyataan penyeleksian struktur switch digunakan untuk memilih salah satu dari banyak cabang blok kode alternatif secara efisien. Sangat ideal dipakai untuk menggantikan tumpukan logika if...else yang bercabang terlalu banyak agar kode lebih mudah dibaca.

Syntax Kode:
int hari = 2;
switch (hari) {
case 1: System.out.println("Senin");
break;
case 2: System.out.println("Selasa");
break;
default: System.out.println("Hari Lain");
}
Output Terminal:
Selasa

While Loop Java

Struktur perulangan while berfungsi untuk mengulang eksekusi sekelompok baris kode secara terus-menerus selama kondisi logika syarat yang ditentukan di awal pemeriksaan selalu terpenuhi atau bernilai benar (*true*).

Syntax Kode:
int i = 1;
while (i <= 3) {
System.out.println("Perulangan ke-" + i);
i++;
}
Output Terminal:
Perulangan ke-1 Perulangan ke-2 Perulangan ke-3

For Loop Java

Perulangan for sangat ideal digunakan apabila Anda sudah mengetahui dengan pasti berapa kali jumlah perulangan baris instruksi tersebut ingin dijalankan. Menyusun inisialisasi, syarat batas perulangan, dan ekspresi aritmatika penambahan dalam satu baris terpadu.

Syntax Kode:
for (int j = 1; j <= 3; j++) {
System.out.println("Angka: " + j);
}
Output Terminal:
Angka: 1
Angka: 2
Angka: 3

Break & Continue

Pernyataan break digunakan untuk menghentikan paksa putaran siklus perulangan secara instan dan langsung keluar dari blok loop berjalan. Sedangkan pernyataan continue digunakan untuk melompati satu iterasi putaran saat ini dan langsung meloncat ke putaran siklus berikutnya.

Syntax Kode:
for (int k = 1; k <= 5; k++) {
if (k==3) {
break;
// Berhenti total ketika k bernilai 3
}
System.out.println("Nilai K: " + k);
}
Output Terminal:
Nilai K: 1 Nilai K: 2

Array Java

Array adalah variabel khusus yang digunakan untuk menyimpan sekumpulan data dengan tipe data yang sejenis dalam satu nama wadah tunggal. Pengaksesan data elemen di dalam susunan struktur array dilakukan dengan memanggil nomor urutan indeks elemennya yang selalu dimulai dari angka nol (0).

Syntax Kode:
String[] mobil = {"Volvo", "BMW", "Toyota"};
System.out.println("Mobil pertama: " + mobil[0]);
Output Terminal:
Mobil pertama: Volvo
